# GNDO Regu-Shield (MVP) 🛡️

**AI-Driven Nuclear Regulation Crosswalk & Design Freeze Simulator**

GNDO Regu-Shield는 한국 KINS/한수원 표준 스펙과 미국 NRC(Part 52 설계인증) 규제 간의 Gap을 교차 분석하고, 설계 변경 시 발생할 수 있는 공기 지연(Schedule Overrun) 및 금융 리스크를 실시간으로 시뮬레이션하는 전문가용 통제 OS 프로토타입입니다.

## 🚀 Key Features
- **Semantic Crosswalk API:** 한국형 SMR 설계 요건과 미국 NRC 규제 조항 조응성 매핑
- **Design Freeze Alert:** 허용정비시간(AOT) 및 설계 기준 위반 시 실시간 하드웨어 록(Lock) 경고
- **Risk Cost Simulator:** 규제 병목으로 인한 예상 공기 지연 및 금융 오버런 비용 가시화
